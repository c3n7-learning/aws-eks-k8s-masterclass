# Introduction

Instructor: Kalyan Reddy Daida

We will cover:

1. Kubernetes services including:
   - Architecture
   - Pods
   - Replica Sets
   - Deployments
   - etc
2. AWS Services including:
   - EKS
   - RDS
   - Fargate
   - etc
3. The intersection of DevOps and AWS Developer Tools
   - AWS CodeCommit
   - AWS CodeBuild
   - AWS CodePipeline
4. Microservices:
   - Service Discovery
   - Distrubuted Tracing
   - Canary Deployments

We will also learn:

5. Docker fundamentals
6. Kubernetes fundamentals: Imperative (with kubectl) & Declarative (yaml)

Database fundamentals:

7. Database with EKS Storage with EBS CSI Driver
8. AWS RDS Database

Networking with:

9. AWS EKS Network Design with EKS Workloads DS & ELB Load balancer
10. AWS Ingress

In addition:

11. AWS Fargate (Serverless)
12. AWS Elastic Container Registry

DevOps:

13. Stages in a release cycle
14. AWS Developer tools and Code Services
15. Microservices with service discovery, distributed tracing and canary deployments
16. Tracing with Kubernetes DaemonSets with XRay pods
17. Horizontal Pod Autoscalers
18. Container Insights via CloudWatch and Fluentd
